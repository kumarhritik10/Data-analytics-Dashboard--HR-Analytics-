# Data-analytics-Dashboard--HR-Analytics-
## Project Objective
To analyze employee data and identify key factors influencing employee attrition and job satisfaction.
To provide actionable insights through interactive dashboards that support effective HR decision-making and workforce planning.

# Dataset used
- <a href="https://github.com/kumarhritik10/Data-analytics-Dashboard--HR-Analytics-/blob/main/HR%20ANALYTICS.xlsx">Dataset</a>

## Questions (KPIs)
1.	What is the total number of employees in the organization?
2.	What is the overall attrition rate?
3.	How many active employees are currently working?
4.	Which department has the highest attrition?
5.	Which job role experiences the most attrition?
6.	Which age group shows the highest employee turnover?
7.	Does education level impact attrition?
8.	Is attrition higher among male or female employees?
9.	How does marital status affect attrition?
10.	What is the average job satisfaction rating of employees?

## Data Process
1.	Data Collection – Gathered employee HR data.
2.	Data Cleaning – Handled missing values and standardized fields.
3.	Data Transformation – Created KPIs and grouped data by categories.
4.	Dashboard Creation – Built interactive visuals for analysis.
5.	Insights & Decisions – Identified attrition trends to support HR decisions.
